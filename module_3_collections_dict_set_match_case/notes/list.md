### Features of good list

#### To declare, you use bracket `[]`

#### Use function list(), `Ex. list(range(0,3)) -> [0, 1, 2]`

#### Indexing -> `Ex. name[0]`

#### The order is preserved (Kolejność jest zachowana)

### MUTABLE!! - you can append, remove arguments

#### Example:
```python
fruit = ['mango', 'apple']
fruit[0] = 'orange'
fruit.append('banana')

fruit.remove("apple")
del fruit[1]
```
