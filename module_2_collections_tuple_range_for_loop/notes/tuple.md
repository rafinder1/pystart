### Features of good tuple

#### To declare, you use bracket
#### Use function tuple(), `Ex. tuple(range(0,3)) -> (0, 1, 2)`
#### Indexing -> `Ex. name[0]`
#### The order is preserved (Kolejność jest zachowana)
### IMMUTABLE!!


#### Example:
```
personal_info = "Rafał", "Protasiuk"
children = (1, 2, 3)
cities = ("Gdynia",)
```

### You do not do that: (IT IS WRONG!!!!!)
```
children[0] = 10
```
